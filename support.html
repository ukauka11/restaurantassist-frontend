What to include

So we can help fast
  • Your business_id (the business name you entered)
  • The email you used at checkout
  • What page you were on (Home / Dashboard / Checkout / Success)
  • If possible: paste the exact error message

Quick help (most common)

2-minute fixes
  • I didn’t get my API key email. Check Spam → click “Not spam”. Then go to Dashboard and paste the key.
  • Dashboard says “401 Unauthorized”. Your key is missing or incorrect. Paste the key exactly (starts with cust_).
  • Dashboard says “402 Subscription inactive”. Subscription isn’t active (payment failed or canceled). Reply to your key email or contact support.
  • “Network error”. Check the API Base URL in the Dashboard (leave it as the default unless you changed it).
