FAQ

Common questions
Where do I find my API key?

After checkout, we email your API key to the email you used. Copy it, then paste it into the Dashboard. You can click Save key locally to remember it on that device.

The Dashboard says “Not connected”. What do I do?
  • Make sure the API key is fully pasted (no extra spaces)
  • Click Load my account
  • Hard refresh the page (Ctrl + Shift + R)
  • If you just paid, wait 30–90 seconds and try again
Why is Generate disabled / not working?
  • You must have a valid API key pasted
  • You must paste a customer review
  • If billing is inactive, requests can return 402
Do you post automatically to Google/Yelp/Facebook?

Right now, the MVP is “copy/paste”: we generate your reply, you post it. Google Autopilot is planned as an optional add-on (connect once).

Do you store my reviews?

RestaurantAssist uses the review text to generate a reply. If we store anything for product improvement, it will be limited and used responsibly. If you want deletion help, email support.

Billing help

Plans & status

If you see a 402 error

402 usually means the subscription is inactive (past_due/canceled). Email support and include:

  • Your checkout email
  • Your business ID (if you know it)
  • What you’re trying to do (ex: Generate reply)

Want to upgrade / cancel?

Email support and we’ll help immediately. Billing portal access will be added to the Dashboard soon.
