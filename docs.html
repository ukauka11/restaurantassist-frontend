Quick Start (non-technical)

Recommended
  • Step 1: Go to Home and click Start free trial.
  • Step 2: After checkout, check your email for your API key (it starts with cust_).
  • Step 3: Go to Dashboard and paste the key.
  • Step 4: Paste a review and click Generate response.
  • Step 5: Click Copy reply and paste it into Google/Yelp/Facebook.
If your email goes to Spam, click “Not spam” so your inbox learns it’s safe.

Help (common issues)

Fast fixes
  • 401 Unauthorized — your key is missing or incorrect (must start with cust_)
  • 402 Subscription inactive — payment failed or subscription canceled
  • Network error — check your internet or the API Base URL
  • Email went to Spam — click “Not spam” (this usually fixes it)

Endpoints

Core API routes
  • GET /health — quick uptime check
  • GET /me — business + plan + subscription status
  • POST /analyze — send review, get AI response + tags
  • GET /summary — summary of saved reviews
If subscription is not active, protected routes return 402.

Headers

Required for protected routes
x-api-key: cust_...
Your API key identifies your business. Keep it private.

POST /analyze body

JSON payload
{
  "review_text": "Food was great but the wait was long.",
  "platform": "google",
  "customer_name": "Optional",
  "order_number": "Optional",
  "run_id": "Optional"
}
platform can be google, yelp, facebook, or other.

What you get back

Response contains a ready-to-post reply
{
  "business_id": "your-business",
  "record": {
    "sentiment": "negative",
    "urgency": "medium",
    "category": "speed_wait_time",
    "tags": ["wait_time"],
    "reply": "Thanks for your feedback! ...",
    "next_steps": ["Review staffing at peak hours", "..."]
  }
}
